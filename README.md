# cascade# Auto Glass Inventory Reconciliation Tool

A high-performance, privacy-focused web tool for reconciling shop inventory against supplier PDFs.

## Features
- **Local Processing:** PDFs are parsed in the browser. No data ever leaves your computer.
- **Three-Way Matching:** Compares tech lists against Bulk, Pending, and Return stocks.
- **Space Gray Theme:** Optimized for low-light shop environments.
- **Instant CSV Export:** Download reconciliation results for Excel.

##  Technical Stack
- **UI:** HTML5, Tailwind CSS
- **Logic:** Vanilla JavaScript (ES6+)
- **PDF Engine:** PDF.js (Mozilla)
